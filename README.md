# Priya-Kumari-JavaStrings
Length() --> This method provides the total count of characters in the string.
charAt() --> This method returns the character at ith index.
substring() --> This method return the substring from the ith index character to end.
concat() --> This method appends the given string to the end of the current string.
toUpperCase()  --> This method converts all the characters in the String to upper case.
toLowerCase() --> This method converts all the characters in the String to lower case.
